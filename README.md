# MXene QC TEM 5ROI

Reproducibility package for the manuscript **Resolving the MXene paradox by microscopy-conditioned variational quantum simulation**.

This repository converts a 200 kV, 500 kX TEM image of Ti3C2Tx MXene into five ROI-conditioned structural motifs, maps FFT-derived priors to effective two-qubit Hamiltonians, solves each motif by exact diagonalization and VQE, performs beta-dependent patch-ensemble reweighting, and now includes a classical TEM/FFT local-ordering-index (LOI) control analysis for direct comparison with the quantum motif hierarchy.

## Core equations

For motif k, the effective Hamiltonian is

H^(k) = -t_parallel^(k)(X⊗I + I⊗X) - t_perp^(k)(X⊗X) + Delta^(k)/2 (Z⊗I - I⊗Z).

The in-plane hopping is mapped from FFT-derived spacing as

t_parallel^(k) = t0 d0 / d_parallel^(k).

Boltzmann motif probabilities are

P_k(beta) = w_k exp[-beta E0^(k)] / sum_j w_j exp[-beta E0^(j)].



## Classical LOI control analysis

In addition to the microscopy-conditioned quantum workflow, this updated package includes a classical TEM/FFT local-ordering-index (LOI) analysis for the same five MXene ROIs. This analysis provides an image-derived control for the quantum motif hierarchy.

The LOI workflow extracts four ROI-level descriptors:

- `P_FFT`: FFT peak-to-background ratio.
- `A_off`: normalized off-center autocorrelation amplitude.
- `C_loc`: local orientation coherence from the structure tensor.
- `Delta_q_eff`: effective FFT peak width, inverted during LOI construction because narrower reciprocal-space features indicate stronger local coherence.

The LOI is calculated after min-max normalization:

```text
LOI = 100 * mean(P_FFT_norm, A_off_norm, C_loc_norm, 1 - Delta_q_eff_norm)
```

A regularized geometric score is then used to obtain LOI-derived patch weights:

```text
g_k = product(descriptor_norm_k + eps)^(1/N)
w_k = g_k / sum(g_k)
```

The script is available at:

```text
scripts/05_classical_loi_analysis.py
```

The processed outputs are stored in:

```text
data/processed/classical_loi/
```

and the generated figure panels are stored in:

```text
data/figures/classical_loi/
```

## Data provenance

Included reproducibility data:

- `data/raw/OneView_200kV_500kX_36135.tif`: raw TEM image used for ROI/FFT analysis.
- `data/processed/roi_parameters.csv`: five-ROI structural and Hamiltonian parameters.
- `data/processed/ensemble_averages.csv`: beta-dependent patch-ensemble averages.
- `data/processed/energy_uncertainty.csv`: uncertainty estimates propagated from TEM/FFT calibration.
- `data/processed/classical_loi/mxene_classical_descriptors.csv`: classical LOI descriptor table.
- `data/processed/classical_loi/mxene_quantum_reference.csv`: quantum reference table used for LOI-energy comparison.
- `data/processed/classical_loi/mxene_ensemble_summary.csv`: equal-area and LOI-weighted ensemble averages.

The submitted manuscript, figures, and Supporting Information DOCX files are intentionally excluded during peer review to avoid version conflicts. They can be added after publication if desired.
